What is HTML?

HTML stands for HyperText Markup Language. HTML is the language that is used to create and display web pages. All web sites, from an individual blog to Microsoft.com are created by writing HTML.

How to code in HTML

HTML is built on tags, like this:

<p>Hello World!</p>

In a web browser, like Edge or Chrome, you do not see the tags:

Hello World!

Here, we used the paragraph ( <p> ) tag. The browser sees the start paragraph tag ( <p> ) and end tag ( </p> ), and understands that Hello World is a paragraph.

Everything you want to display on a web page needs to be enclosed in html tags. So let us look at the basic tags we need to display a web page.

The basic tags of all web pages

Here we are going to learn the coding tags that are required for all web pages:

<!DOCTYPE>

<html>

<head>

<body>

Every HTML document starts with the !DOCTYPE tag:

<!DOCTYPE html>

Note: The !DOCTYPE tag does not need an end tag.


The next required tag is the html tag. Everything you want to display on the page), goes between the start and end html tags:

<!DOCTYPE>

<html>

</html>

Web page title

The <head> tag contains the page title:

<head>

<title> The title of this page </title>

</head>

Body

The <body> tag holds the web content.

Putting it all together

Combine all these tags, we have a simple HTML file that will display in any web browser:

<!DOCTYPE html>

<html>

<head>

<title>Lesson 1: What is HTML?</title>

</head>

<body>

<p> HTML stands for HyperText Markup Language. </p>

</body>

</html>

Things to remember:

HTML tags are not displayed when your code is processed in the web browser. That is because the tags are read by the browser so it will know what kind of content you have written.
