Lesson 3: HTML Structure

To write good HTML code, it is important to understand structure. Structure elements allow you to organize the main parts of your page.

Grouping elements together

When writing HTML content like paragraphs, lists or links, you provide meaning to your text. But you might want to group some of these elements together.

For example, a simple web page can be divided in parts:

  • A header, the is the first element on the page and contains similar on every page
  • Main content, that changes for every page: a list of articles, a single article with comments, an about page, etc
  • List of links to other pages of the web site
  • A footer for additional links to less important pages

These parts are defined with the following tags:

  • <h1> as the title of the page.
  • <article> as the main content of the page, like a blog post.
  • <footer> as the last element of the page, located at the bottom.

Adding a header

The header is usually the first HTML element in the code. Use it to add images, text, and links you want at the top of the web page.

<header>

<p>

<a>

<img src="my-logo.jpg" alt="CodeUnbarred logo">

</a>

<em>A cool website</em>

</p>

</header>

Main content

The main element contains, as its name suggests, the most important content of the page, the one that defines the purpose of the page. While most webpages of a website contain common elements (like the header, the navigation, the footer), the main element focuses on unique content.

Headings

There are 6 levels of headings available, ranging from <h1> to <h6>, 1 being the most important one.

Headings are used to create a natural structure to your document.

Only using headings in a HTML document would not make sense: if everything feels important, nothing really is. You need to provide depth in your structure.

The Footer

The footer is usually the last element of a page, where the main navigation links are repeated and secondary ones added.

Footer Example

<footer>

<p>MarkSheet.io | Copyright 2015</p>

<ul>

<li>

<a href="home.html">Home</a>

</li>

<li>

<a href="about.html">About</a>

</li>

<li>

<a href="contact.html">Contact</a>

</li>

</ul>

<ul>

<li>

<a href="privacy-policy.html">Privacy Policy</a>

</li>

<li>

<a href="terms-of- use.html">Terms of use</a>

</li>

</ul>

</footer>
