Working with Text

CSS provides several font properties, that directly affect text appearance:

The <font-family> property defines which font to use.

The <font-size> property defines what size font to use.

The <font-weight> property can make your text bold .

The <font-style> property can make your text italic .

Along with these font properties, CSS also has many properties for editing the appearance and alignment of the text itself:

The <text-align> property must be applied on a block-level element and defines how its text and children inline elements are horizontally aligned.
The most used values are: left, right and center.

The <text-decoration> property is used to add a line on your text.
Most commonly used values are none (default), line-through, underline, and overline.

The <text-indent> property allows to add space before the first letter of the first line of a block-level element.
