Where do I write CSS?

There are a few ways you can write CSS to go with your HTML:

You can add a 'style' attribute into an HTML tag

<p> style = "color: red;"> This text is important. </p>

You can include <style> in the <head> of your HTML document.

<html> <head> <title> Hello World </title> <style type="text/css"> p{ color: red;} </style> </head> </body> <p> This paragraph will be red. </p> </body> </html>

You can link a separate CSS page to your HTML document in the <head>.

CSS file:

p{color: red;}

HTML file:

<html> <head> <title> Hello World </title> <link rel="stylesheet" type ="text/css" href="style.css"> p{ color: red;} </head> </body> <p> This paragraph will be red. </p> </body> </html>

The third method is most commonly used, and it is the method we will be using to add CSS to our HTML web pages.
