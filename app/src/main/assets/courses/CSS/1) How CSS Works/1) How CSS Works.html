How CSS Works

CSS works by selecting an HTML element (like a paragraph), choosing a property to alter (like the color), and applying a certain value (like red):

Example: p {color: red;}

CSS is a 3-part process that contains:

This whole block (selector/property/value) is a CSS rule.

selector{property: value;} or you can think of it as who{ what: how;}

The selector defines who is targeted, which HTML element(s)

The property defines what characteristic to alter

The value defines how to alter that characteristic

A CSS can have multiple selectors and multiple properties.

CSS Selectors: How to Target HTML Elements

Because we do not want to style all our HTML elements at once, we need to be able to target a group of these HTML elements.

CSS selectors define which elements we want our styling to be applied to.

There are many different ways to select various HTML elements:

Generic HTML Tags - Targeting generic HTML tags is easy: just use the tag name (like p, a, img, ul, li, etc)

In CSS:
a{ /* Links */ }
a{ /* Paragraphs */ }
a{ /* Unordered Lists */ }
a{ /* List Items */ }

HTML Classes - if you do not want to style all paragraphs or all titles identically, we need to differentiate them by using HTML Classes.

Of all HTML attributes, the class attribute is the most important for CSS. It allows us to define a group of HTML elements that we can target specifically. Just put a dot . in front of the class name you want to use:

In CSS:
.date{
color: red;
}

In HTML: <p class="date">
Saturday Feb 21
</p>

HTML IDs - You can also use the id attribute in your HTML, and target it with a hash # in your CSS:

In CSS: #tagline{ color: orange;}

In HTML: <h1 id="tagline"> This heading will be orange.</h1>

Combined selectors: let us say you wanted to select in CSS all the paragraphs with a main class. To select these elements, you would use p.main

Parent-Child selectors - A space in a selector defines a ancestor/descendant relationship. Let us say we want the links in our header to be in red:

header a {
color: red;
}

This can be read from right to left as: Select all a elements that are within a header element. This will prevent all other links (that are not in the header) from being changed

You can use the CSS body selector to modify the style of your overall web page. This is helpful if you want all the text in the page to have the same font styles, or if you want to add a background color or image to your overall page.
