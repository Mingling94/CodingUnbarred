Lesson 2: Adding text, pictures, and links to a web page

Depending on the content you’re writing, you can choose HTML elements that match the meaning of your text.

Adding text to a page

We have already used the paragraph <p> tag. Let us look at a few more ways to define your content:

  • <p> for paragraphs
  • <ul> for bulleted (unordered) lists
  • <ol> for numbered (ordered) lists
  • <li> for list items

Lists

To create a list, you use the following code:

<p><b>Types of lists</b></p>

<ul>

<li>numbered (ordered) use the <ol> tag </li>

<li>bullet (unordered) lists use the <ul> tag </li>

</ul>

The <ul> tag tells the browser it is a list, and the <li> tag tells the browser where each individual item in the list starts and ends

Note that each <li> tag is closed before the next one starts. Also note that the <li> and </li> tags are enclosed within the <ul> and </ul> tags. This is called nesting. Nesting is very important when writing HTML code. A child element, like <li> must be closed before closing the parent element <ul>.

Adding images to a page

The image tag <img> is different from the previous tags in two ways:

  1. It is a self-closing element (it only has an opening tag).
  2. It needs more information, to make it work. For example, you need to include attributes such as the name of the image, where it is located, and its size.

  • Image location: The src attribute defines the location of the image.

    <img src="soyuz-spacecraft.jpg">

  • Image size: Every image has a width and a height. You do not need to size an image, the browser will display it in full size. But if you do want to change the size, you can use CSS.

Adding Links to Other Pages

The HyperText part of HTML defines the links we use: hypertext links, called hyperlinks. You navigate from one document to another or from one web site to another by clicking on links.

A typical link to another web site looks like this:

<a href="http://www.nasa.gov/”</a>

The href attribute defines the target of the link (where you navigate to when you click).

When you want to link to another web page, you need the complete URL to document:

http://nasa.gov

No matter where you use the link, it contains all the information required to find the correct file, on the correct domain, with the correct protocol. You usually use absolute URLs defining a link from your website to another website.

<p>

Find more images of space on <a href="https://nasa.gov"> </a>!

</p>
