HTML

What is HTML?

HTML stands for HyperText Markup Language. HTML is the language that is used to create and display information on the Internet. All web sites, from an individual blog to Microsoft.com are created by writing HTML.

Why you need it?

Web browsers, like Internet Explorer, Chrome, and FireFox, read HTML and interpret the code to display information.

What you'll learn

In these lessons, you will learn the basics of writing HTML code that can be read by a web browser.

How long will it take?

Each lesson should take about 20 minutes. Writing sample code may take longer.
