Intro to CSS Lesson:

What is CSS?

While HTML describes the content of a web page, CSS changes the appearance of a web page.

CSS stands for Cascading Style Sheets. Its purpose is to style a markup languages (like HTML). Therefore, CSS is only useful when it is paired with an HTML document.

CSS brings an HTML document to life by choosing fonts, applying colors, defining margins, positioning elements, animating interactions, and much more.

What you will learn in this section:

In Lesson 1 you will learn the basic parts of CSS that make it work.

In Lesson 2, you will learn where you add your CSS code in an HTML document.

In Lesson 3, you will learn about the CSS code you can use to change the way the text of your HTML document looks.
